agoras.media package
===================

The ``agoras.media`` package provides media download and processing capabilities for images and videos.

Package Overview
----------------

This package handles:
- Image media files (JPEG, PNG, etc.)
- Video media files with platform-specific size limits
- Media factory for creating and managing media instances

.. note::
   **Package Dependencies:** This package depends on ``agoras-common`` for utilities and logging. See :doc:`common` for details.
